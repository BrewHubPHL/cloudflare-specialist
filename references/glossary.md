# Glossary

| Term | Definition |
|------|------------|
| **workerd** | Cloudflare Workers JavaScript/Wasm runtime |
| **Binding** | Named handle on `env` to KV, R2, DO, etc. |
| **wrangler** | CLI for dev, deploy, secrets, types |
| **cloudflared** | Daemon for Cloudflare Tunnel outbound connections |
| **Access** | Zero Trust authentication layer in front of apps |
| **Service token** | Machine identity for Service Auth policies |
| **Hyperdrive** | Connection pooler between Workers and Postgres |
| **OpenNext** | Build adapter for Next.js on edge runtimes |
| **DO** | Durable Object — strongly consistent edge state |
